Instructions de déploiement sur Vercel :
1. Rendez-vous sur https://vercel.com
2. Cliquez sur “+ Nouveau projet” ou “Add New…”
3. Choisissez “Importer un projet manuellement” (sans GitHub)
4. Glissez-déposez ce dossier décompressé dans l’interface
5. Cliquez sur Déployer – Vercel génèrera une URL publique.
